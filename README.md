# تطبيق تعليم الأطفال العراقيين
**Educational App for Iraqi Children (6–9 years)**

---

## 📋 نظرة عامة

تطبيق Android تعليمي تفاعلي يقدم محتوى تعليمي باللهجة العراقية للأطفال من عمر 6–9 سنوات.

**الميزات الرئيسية:**
- 30 درس تفاعلي (لغة عربية، رياضيات، علوم)
- أنشطة ممتعة (سحب، مطابقة، ألعاب)
- يعمل بدون إنترنت
- نظام نجوم ومكافآت
- واجهة عربية RTL
- محتوى باللهجة العراقية

**المدة:** 4–6 أسابيع  
**الإطلاق المستهدف:** أواخر أبريل 2026  
**المنصة:** Android (API 24+)، iOS لاحقًا
