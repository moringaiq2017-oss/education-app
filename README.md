# تطبيق المُعَلِّم
**Al-Muallim App — مساعد الأم الذكي لتعليم أبنائها**

---

## الفكرة

تطبيق موجّه **للأمهات العراقيات** لحل المشاكل اليومية مع أبنائهم بالمرحلة الابتدائية. التطبيق يساعد الأم بتدريس ابنها بطريقة تفاعلية ذكية بدل ما تعاني بالشرح والتكرار كل يوم.

**المشكلة:** الأم تقضي ساعات يومياً بتدريس ابنها — إملاء، حفظ، جدول ضرب، إنكليزي — والنتيجة تعب وأعصاب.

**الحل:** تطبيق يتولى التدريس التفاعلي بدلاً عنها بطرق ممتعة للطفل.

---

## المواد الدراسية — الصف الأول ابتدائي

| المادة | المحتوى | الحالة |
|--------|---------|--------|
| الإسلامية | 18 درس: سور قرآنية + أحاديث + آداب + سيرة نبوية | مكتمل |
| القراءة العربية | إملاء تفاعلي + أناشيد بالكاريوكي + محفوظات | مكتمل |
| الإنكليزي | نطق + كتابة الكلمات | قيد التطوير |
| الرياضيات | جمع وطرح بطرق مبسطة | قيد التطوير |
| العلوم | شرح وفهم (مو حفظ) | قيد التطوير |
| الأخلاقية | دروس تفاعلية | قيد التطوير |
| الألعاب الذهنية | تركيز وذاكرة | قيد التطوير |

---

## المميزات

- **إملاء تفاعلي** — التطبيق ينطق الكلمة والطالب يكتبها
- **أناشيد بالكاريوكي** — النص يتلون مع الأغنية كلمة بكلمة
- **محفوظات بـ 3 مراحل** — اسمع وتابع → أكمل الناقص → اختبر نفسك
- **صوت طبيعي (Google TTS)** — صوت أنثوي HD للنطق
- **سور قرآنية** — عرض بتصميم ذهبي مع شرح مبسّط
- **أحاديث نبوية** — حفظ + شرح + أسئلة مناقشة
- **تعليم إلكتروني** — قسم لرفع دروس فيديو داخل كل مادة
- **تصميم عصري** — ألوان بنفسجية، كاردات مدورة، واجهة RTL

## تشغيل المشروع

```bash
# 1. الباكند
cd backend
pip3 install -r requirements.txt
python3 -m uvicorn app.main:app --reload --port 8000

# 2. التطبيق
cd mobile
flutter pub get
flutter run
```
