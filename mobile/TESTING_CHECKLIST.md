## 🔌 اختبارات API Integration

### Endpoints:
- [ ] POST /auth/register - تسجيل يعمل
- [ ] POST /auth/login - تسجيل دخول يعمل
- [ ] GET /tracks - جلب المسارات يعمل
- [ ] GET /tracks/:id/lessons - جلب الدروس يعمل
- [ ] POST /progress - تحديث التقدم يعمل
- [ ] GET /children/:id/progress - جلب التقدم يعمل
- [ ] GET /children/:id/achievements - جلب الإنجازات يعمل
- [ ] GET /children/:id/stats - جلب الإحصائيات يعمل

### Error Handling:
- [ ] عند قطع الإنترنت: يظهر رسالة خطأ مفهومة
- [ ] عند إيقاف Backend: يظهر رسالة "لا يوجد اتصال"
- [ ] عند خطأ 500: يظهر "خطأ في الخادم"
- [ ] عند خطأ 404: يظهر "البيانات غير موجودة"
- [ ] Timeout يعمل بعد 30 ثانية
